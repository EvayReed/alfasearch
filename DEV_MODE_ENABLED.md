## 🔄 如何禁用开发模式

当 Google 配置修复后，只需在 `AuthContext.tsx` 中注释掉开发模式代码：

```typescript
// 注释掉这个条件判断
// if (__DEV__) {
//   // 开发模式代码...
//   return;
// }
```
